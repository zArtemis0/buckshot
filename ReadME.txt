DefenderKiller - утилита с открытым исходным кодом для удаления антивируса Windows Defender и его компонентов (службы, задачи, драйвера, папки, приложение Безопасность Windows и SmartScreen)

В папке Work находятся рабочие файлы программы. Запускать оттуда что-либо не требуется.

В утилите имеется возможность восстановить ранее удалённый защитник, если Вы в процессе удаления создали его резервную копию. С помощью этой резервной копии позже можно восстановить защитник Windows в полностью исходное состояние, если он Вам опять понадобится. 

Удаление защитника Windows может конфликтовать с работой центра обновлений Windows (с обновлениями), поэтому, на всякий, перед удалением создайте резервную копию, если Вы обновляете Windows во избежание последующих проблем с этим центром.
